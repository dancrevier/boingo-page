Daily Kanji is a Windows Phone application that displays a different kanji (Japanese character) every day.

Version 1.0 supports the following features:

  • Live Tile updates
    • Note: You must launch the application every 10 days to update tiles due to Windows Phone restrictions
  • Select the range of kanji to use based on Japanese Language Proficiency Test (JLPT) level.
