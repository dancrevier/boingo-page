That is the basic pattern used for all the choruses, but I just used the first choruses, because that’s where it first comes in (and I like that chorus).
The base for this song is on a six string with a pedal (?), but the rhythm guitar can help it along by using this stroke (strum) muted or dead stringed.
You go: Down   Down   Down Up Down
	 Down   Down Up Down Up Down Up Down
Listen to the song’s base line and you should understand me. You might find it easier to just do this:

Lead guitar should just use a lot of distortion and eerie pitch bending effects during the verse.

Sure this tab is long and complex, but take the time to learn it, I’m sure it’s correct.
Have fun playing!
